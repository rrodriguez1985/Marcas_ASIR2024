Introducción a Lenguaje de Marcas


Definición de página web: es un documento que contiene texto con etiquetas y marcas.
Las marcas son unas etiquetas, que nos van a permitir insertar y mostrar información.
Vamos a poder mostrar texto, imágenes, audios, vídeos, etc.
¿Cómo podemos visualizar las páginas webs?

Definición de HTML

Se entiende por HTML ( HyperText Markup Language), es decir, "Hipertexto de Leguaje de Marcado"
Se caracteriza por ser un lenguaje de marcas o etiquetas, que se utilizan para poder diseñar / crear páginas web.
El lenguaje HTML, está definido por el Word Wide Web Consortium (W3Consortium)
Permite aglutinar texto, imágenes, audios, vídeos, etc. y combinarlo a nuestro gusto.
Es un estandar de visualización de información.


Etiquetas en HTML

Para crear una etiqueta en HTML, tenemos que tener en cuenta lo siguiente:
  • Tendremos una etiqueta de apertura que irá con el simbolo de mayor qué y el nombre de la etiqueta
    seguido de una etiqueta de cierre
  • Irá el código/información que queremos que vaya asociado a dicha etiqueta
  • Tendremos una etiqueta de cierre que estará formada por el simbolo de mayor qué, una contrabarra
    y el nombre de la etiqueta seguido del cierre.



    • Esto es un ejemplo de párrafo con etiqueta de apertura y cierre

    Recordar, que la etiqueta de cierre tiene siempre que ir con una barra del /



    En HTML, la gran mayoría de etiquetas tienen una etiqueta de apertura y otra de cierre.
    Eso no quita, que exitan algunas etiquetas especiales, que sólo tiene etiqueta de apertura.
    Estas etiquetas son las que se conocen como monoetiquetas, y algunas de las más importantes o más usadas son las siguientes:
    img (para imágenes), hr (para establecer una línea de separación horizontal), br (para establecer salto de línea), entre otras.

Cómo poner comentarios en HTML

Un comentario en HTML, sirve para dar información a un programador o persona que va a trabajar con nuestro documento HTML.
Para poder poner un comentario en HTML, utilizaremos la siguiente estructura:
  • Tendremos una etiqueta de apertura que tendrá un signo de mayor qué, seguido de un símbolo de
    exclamación de cierre seguido de dos guiones ( < ! --)
  • Posteriormente, se introduciría el texto del comentario a indicar
  • Se cierra con una etiqueta que tiene la siguiente estructura: -- >

Cuando vayamos a crear un documento HTML, es importante tener en cuenta que la extensión con la que
vamos a guardarlo será con .html
Si no lo guardamos con esta extensión, NO SE CONSIDERA UNA PÁGINA WEB, Y NO SE EJECUTA EN EL NAVEGADOR.

Estructura de un documento HTML

Los documentos HTML, están divididos en dos zonas princiapales:
  • Cabecera: (head) Es la zona del documento que incluye el título y el idioma entre otros elementos
  • Cuerpo: (body) Incluye todos los contenidos y elementos que formarán nuestra página web: párrafos, texto, imágenes, audios, vídeos, etc.

Formato de textos

  • Etiqueta b: se utiliza para poner el texto en negrita
  • Etiqueta i: se utiliza para poner el texto en cursiva
  • Etiqueta del: se utiliza para tachar texto
  • Etiqueta em: se utiliza para hacer énfasis en un texto
  • Etiqueta small: para hacer textos pequeños
  • Etiqueta sup: se utiliza para poner un superíndice en un valor. X2
  • Etiqueta sub: se utiliza para poner un subíndice en un valor. X2
  • Etiqueta mark: se utiliza para subrayar un valor
  • Etiqueta q: se utiliza para poner

    citas pequeñas en textos

  • Etiqueta cite: se utiliza para poner un título de trabajo
  • Etiqueta time: se utiliza para poner hora 20:00
  • Etiqueta address: se utiliza para poner una dirección:
    calle tajo
  • Etiqueta blockquote: Hacer grupos de etiquetas
